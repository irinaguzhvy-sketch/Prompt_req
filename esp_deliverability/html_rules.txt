HTML & LAYOUT RULES:
- Max width 600px
- Table-based layout only
- No JavaScript
- No external fonts
- No base64 images
- HTML size < ~100 KB
- Include hidden preheader block
- Images: ALT text + explicit width/height
- Avoid !important in inline CSS
- Outlook compatible: Use VML for buttons
- Avoid: background-image, float, position, flex/grid
- Conditional content: simple logic
- Fallbacks for all conditional content
- Logo protection: avoid color conflicts
